Este proyecto es una aplicación web desarrollada con React que permite a los usuarios explorar animes utilizando datos de una API pública. Incluye una página de inicio con tarjetas estilizadas que muestran animes destacados, un buscador interactivo para filtrar contenido por nombre o género, y una sección de recomendaciones. Se han utilizado tecnologías como React Router para la navegación, Bootstrap para el diseño responsivo y efectos visuales, y hooks como useState y useEffect para el manejo dinámico de datos. El diseño está optimizado para ser limpio, atractivo y funcional, con una cuadrícula de tarjetas organizadas en filas de 3, adaptadas a diferentes tamaños de pantalla al igual que se utiliza para los errores "ErrorBoundary".
